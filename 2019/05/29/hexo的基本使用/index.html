hexo的基本使用

​

一:hexo导航菜单

  1. 使用: hexo new page “test”

​ 在目录: Hexo根目录/source/ 下生成了一个文件夹test,文件夹内部有index.md文件,可以进行修改

  1. 修改theme文件夹主题下的配置:默认主题是landscape

    在_config.yml的menu下添加 test:test(符合yaml语法)

  2. 作用:通过导航栏可以通过http跳转页面,也可以做成二级导航形式(下拉)

  3. 启动hexo: hexo s.默认http://localhost:4000/可以访问,在字上面新增了一个test导航

二:hexo生成文章并指定分类,标签

  1. 生成文章

    hexo new “title”

    在目录 Hexo根目录/source/_posts下生成了 title.md

  2. 修改文章头(在title.md开头部分)

    title: 文章的标题
    date: 时间
    tags: test标签(不需创建)
    categories: test分类(不需创建)

    格式符合yaml语法

    这样,这篇文章就属于 test分类 test标签

  3. 文章内容使用markdown语法正常编写

三:高级用法

  1. 切换主题背景

    步骤一:找到喜欢的主题,下载( 推荐资源: https://hexo.io/themes/)

    例如我选择cafe主题:

    在本地hexo博客工程根目录下进入git bash:

    git clone https://github.com/giscafer/hexo-theme-cafe.git themes/cafe

    在themes生成一个cafe目录,内容是下载的主题

    步骤二:设置新主题

    修改hexo博客工程根目录下的_config.yml

    步骤三:启动查看效果

    1. 添加图片

      步骤一:修改hexo博客工程根目录下的_config.yml

      ​ psot_asset_folder:设置为true

      步骤二:安装插件

      步骤三:hexo new “title”的时候会在\source\_posts下生成于文章同名的文件夹

      ​ 内部存放图片,在文章中通过 图片文件夹名\图片名访问

      ​ 如果先生成了文章,后来发现要添加图片,那么可以自己添加一个同名文件夹放图片,使用方法同上
